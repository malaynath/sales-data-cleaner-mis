# Sales Data Cleaner & MIS Report Generator

A Python automation tool that:

* Reads CSV and Excel files
* Removes duplicate records
* Handles missing values
* Detects sales columns automatically
* Generates MIS reports
* Creates Excel dashboards
* Produces sales charts automatically

## Features

* CSV & Excel Support
* Automatic Data Cleaning
* Sales Summary
* Top Product Analysis
* Monthly Sales Trend Analysis
* Excel Report Generation
* Chart Generation

## Technologies Used

* Python
* Pandas
* Matplotlib
* OpenPyXL

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python sales_mis_report.py
```

Enter:

```text
sample_data.xlsx
```

or

```text
sales.csv
```

## Output

* cleaned_sales_report.xlsx
* top_products_bar_chart.png
* monthly_sales_line_chart.png
