# 📊 Sales Data Cleaner & MIS Report Generator

A Python-based automation tool that cleans sales datasets, generates MIS reports, and creates visual analytics automatically.

## 🚀 Features

* Supports CSV and Excel files
* Automatic dataset loading
* Column standardization
* Duplicate removal
* Missing value handling
* Automatic sales column detection
* Product-wise sales analysis
* Monthly sales trend analysis
* Excel report generation
* Automated chart generation

## 📂 Dataset Used

**Online-Store-Orders.xlsx**

The project analyzes sales data from an online store and generates business insights automatically.

## 🛠 Technologies Used

* Python
* Pandas
* Matplotlib
* OpenPyXL
